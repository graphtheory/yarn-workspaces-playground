# Install

Navigate to the root folder and run yarn install

```bash
$ yarn install
```

# Run the app

```bash
$ yarn start
```

# Adding a dependency

Add `webpack` to `app` as `devDependencies`.

```bash
$ yarn workspace app add webpack --dev
```


# Remove a dependency

Remove `webpack` from `app` as `devDependencies`.

```bash
$ yarn workspace app remove webpack --dev
```
